# NEKROMANCER — Rise of the Dead

A Diablo 3 inspired hack-and-slash in the Diablo 4 mobile gameplay style, built for the browser.
Command the **Nekromancer**: shred the horde with bone magic, detonate corpses, and raise skeletal warriors to fight at your side.

## ▶ [Play Game](https://nors3ai.github.io/Nekromancer/)

Runs in any modern browser — best experienced on a phone in landscape, but fully playable on desktop.

## Controls

| | Mobile | Desktop |
|---|---|---|
| Move | Left-thumb virtual joystick | WASD / Arrow keys |
| Bone Splinters (primary) | Big right button (hold) | Space / J / Left click (hold) |
| Bone Spear | Skill button | `1` |
| Corpse Explosion | Skill button | `2` |
| Skeletal Warriors | Skill button | `3` |
| Blood Nova | Skill button | `4` |
| Mute | — | `M` |

All skills auto-aim at the nearest enemy — one-thumb friendly.

## The Nekromancer's kit

- **Bone Splinters** — free primary attack; each shard that connects generates Essence.
- **Bone Spear** — piercing spear that skewers whole packs.
- **Corpse Explosion** — every slain enemy leaves a corpse; detonate up to 5 at once.
- **Skeletal Warriors** — raise up to 4 skeletons that hunt for you.
- **Blood Nova** — eruption that damages, slows and knocks back everything around you, healing and shielding you per enemy hit.

## Gameplay

Endless waves of the risen dead — zombies, skeletons, ghouls, cultists — with a hulking **Brute** boss every 5th wave. Kill, level up, collect gold and health orbs, and see how deep into the night you can survive.

## Tech

Pure HTML5 canvas + vanilla JavaScript. No frameworks, no build step, no assets — every sprite, sound effect and map tile is generated procedurally at runtime. The game lives in [`docs/`](docs/) and is served by GitHub Pages.
